# MONTRACE - Finance Tracking App

MONTRACE is a premium finance tracking application designed to help users manage their income and expenses with ease. The app features a modern, clean interface with a custom design system.

## 🚀 Features

- **Dynamic Dashboard**: View your total balance, income, and expenses at a glance.
- **Transaction Management**: Easily toggle between Income and Expense categories.
- **Interactive Dropdowns**: Dynamic category selection based on transaction type.
- **Onboarding Flow**: Beautifully designed splash and onboarding screens to welcome users.
- **Custom Navigation**: A sleek, bottom navigation bar tailored to the design aesthetic.

## 🛠️ Built With

- **Language**: Java
- **Platform**: Android
- **UI Framework**: XML Layouts with Material Components
- **Design System**: Custom Navy & Purple color palette with rounded UI elements.

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/RJoshuu70/UTSAD.git
   ```
2. Open the project in **Android Studio**.
3. Sync Project with Gradle Files.
4. Run the app on an emulator or physical device.
